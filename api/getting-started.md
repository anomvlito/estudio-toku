### ⏩ Quick Start

Aquí podrás leer en 5-10 minutos lo que necesitarás para estar integrado a Toku. Esta es una guía rápida para poder empezar a funcionar correctamente, esto es, crear clientes, productos y deudas en Toku, con el fin de procesar los pagos de tu organización.

Secciones:
- Backend
- Frontend

### 👩🏻‍💻 Backend

Aquí encontrarás las distintas formas que tenemos para poder integrar la principal información de Toku a través de sus diferentes canales:

- **Data in | Información de la organización a Toku**
  - Via API
  - Via SFTP
  - Carga Manual
  - Formularios en Toku: Para ejecutivos y autogestionado

- **Data out | Información de Toku a organización**
  - Via API
  - Via SFTP
  - Descarga manual

- **Backend: Carga tus datos a Toku y recibe notificaciones**
  - Via API
    - Subir deudas y recibir pagos
    - Inscribir medio de pago automático
  - Via SFTP
    - Subir deudas y recibir pagos
    - Construir archivo de invoices y payments
  - Carga Manual
  - Formularios en Toku: Para ejecutivos y autogestionado
  - Notificaciones y webhooks
